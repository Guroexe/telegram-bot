# 🔧 Исправления для Railway.com - Ошибка при записи на аренду

## Проблема
При записи на дату аренды на Railway.com появляется ошибка:
```
❌ Расписание на этот месяц недоступно! Пожалуйста, выберите другой месяц.
```

## Причины на Railway
1. **Проблема с временной зоной** — Railway использует UTC вместо московского времени (UTC+3)
2. **Листы расписания не создаются** — фоновая задача не работает правильно или стартует поздно
3. **Первый запуск** — листы могут не существовать при первом включении бота

## Внесённые изменения

### 1. ✅ Добавлена поддержка московского времени
- Импортирован `pytz`
- Все функции теперь используют `datetime.datetime.now(MOSCOW_TZ)` вместо `datetime.datetime.now()`
- Это гарантирует что дата и месяц определяются в московском времени

### 2. ✅ Добавлено автоматическое создание листов
- Новая функция `create_sheet_if_not_exists(sheet_name)` создаёт листы по требованию
- Листы создаются автоматически при попытке бронирования
- Если лист не существует, система создаёт его со стандартными заголовками

### 3. ✅ Листы создаются при запуске бота
- Функция `post_init()` теперь сразу создаёт листы на месяц вперёд
- Не нужно ждать 3 часа ночи для первого создания

### 4. ✅ Фоновый планировщик использует московское время
- `background_scheduler()` теперь рассчитывает время до следующего запуска правильно
- Создаёт листы для 4 месяцев вперёд (текущий + 3)

### 5. ✅ Обновлен requirements.txt
- Добавлен `pytz==2024.1`

## Как это работает теперь

1. **При запуске бота** → создаёт листы для текущего месяца и 3 следующих
2. **При выборе даты** → если лист не существует, автоматически создаёт его
3. **При фоновом запуске (каждый день)** → проверяет и создаёт листы на случай если они потеряны

## Тестирование

1. Перезагрузите бота на Railway
2. Попробуйте записаться на любую дату
3. Ошибка "Расписание недоступно" больше не должна появляться

## Если ошибка всё ещё появляется

Проверьте в логах Railway:
- Загружен ли `credentials.json`?
- Есть ли ошибки от Google Sheets API?
- Используется ли правильное время (московское)?
